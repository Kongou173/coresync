# Coresync bot
多機能botです。
現在のbotのバージョンは1.0.0です。

## Botの機能
- ユーザーをkick・BAN、timeoutする。
- Botが代わりにメッセージを送信する
- 現時刻を表示する
- Geminiとチャットをする・チャット履歴を消す
- 情報をWikipediaから取得する
- ランダムに名前を変更する
- 所属しているサーバーの情報を取得する
- 天気情報を取得する
- アキネーターで遊ぶ
- googleの検索結果を取得する

## 実装コマンド
- `/help`
- `/ban <user>`
- `/kick <user>`
- `/timeout <user> <duration>`
- `/support`
- `/say <message>`
- `/time`
- `/chat <message>`
- `/chat_clear`
- `/random`
- `/wiki`
- `/server`
- `/whether`
- `/akinator`
- `/search`
